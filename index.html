语言的极限

发表于 2023-01-17

我已经感受到了自己语言所能描绘的极限。这个极限不是真正意义上的极限，它是一个分界线，在极限以内，我可以自如的得出这些结论、组织这些语言，像喝水一样自然；但是在极限以外，我并不排除我的确也能写出《浮士德》这样的作品，但是它的几率很小，小到约等于零，在组织语言的过程中，我需要大量的思考，本质上就是一种重塑，对我旧有语言体系的推翻，所以它处于极限之外。
我用了十几天的时间，对自己的整个意识系统进行了一番游历，这是一个从看一棵棵树到俯瞰整个森林的转换。这个游历是很有意义的，它使得我对我整体、局部的结构有着一个非常清晰的认识，甚至说，我对我的一些模式都有了更为清晰的模式，当我再一次走进它们，我变得可以意识到我一遍又一遍的走进过这个循环，我可以直接通过记忆而非再进行一遍计算就能得到我想要的结果，按程序语言，这是一种记忆化。
对自己的结构有了更清晰的认识之后，最大的奖励就是我可以对它进行修改创作，我之前的一切都是屎山代码，我现在找到了屎山的核心结构，抖落掉冗余的废物，对那些结构我可以进行修改。
这种体验，不可言喻的体验是孩童般的体验，这种感受是一个刚刚接触学习的孩子的感受，我现在有着一种难以言喻的喜悦，一种遗失了11年的喜悦。
我现在已经不需要说话了，不需要输出了。

时间

发表于 2023-01-14 更新于 2023-01-17

2023.1.14

时间的分割

虽然我是一个讨厌既定章程的人，喜欢生活过得更加随性所居一些，但是，我总是喜欢一个任务在一个完成的时间块儿内被完成。有时为了这种不可改变的固执，我恨不得从早上八点一直做到晚上八点。
时间的分割，是因由这个现象而诞生的。它阻碍了我这样的进程。它可以是我必须要吃饭，我必须要上厕所，为了别人的方便，我必须在什么时间完成一件事情；以及我必须要睡觉，而且我的生物钟命令我在一个时间范围内睡觉。
这已经是我可以接受的范围了，也是我能接受的极限了。
真正给我的世界带来恶心的感觉的是四中，在那里的六年，我的时间被切成一个又一个细小的碎片，为了在每一个碎片内完成他们分配给你的任务，你必须要按照他们的程序去思考。即便我反抗，即便我什么都不做，但是我还是在最根本最深层上无法逃离这个定律。我可以轻描淡写地说“四中是狗屎，体制内是狗屎，这不是我要的生活”。
我的一些矛盾的症结所在便是，我抵触时间的分割，但是这六年又致使我的行动都在依赖这种对时间的分割，这种矛盾，对峙，非常的内耗。
时间，思维进程，[[能量]]，它们三者环环相扣。
似乎，我对时间分割的讨论没有多么深刻，其中满满的是一种抱怨。

时间的本质探讨

比起时间，我根本不在乎时间的流逝，我也并不常以时间的线性流逝去考虑问题；对我而言，对目前的我而言，时间似乎仅仅是能量流逝的一种刻画，一种计量。而且客观意义上，我经受不起漫长的等待，我的意识是不断进化的，我并不像某些人一样秉承着一些比较浅显的信条然后坚信（我没有不尊重他们，他们根据这个信条建立起的世界也是伟大的，只是我们观念不同）。（我的意识此时似乎想要逃避，想要掩盖什么马上就可以被推导出的真相，它经常这样，每当此时它都会劝我拿起手机，刷一些无聊的内容，把自己意识丢掉，做一些机械的事情）。
因为编程，时间给我的一种感受是它是一种限制，一种你必须面对的事实、规则，对，它是这个宇宙的规则，你必须面对它。

我认为我需要引进一个新的词条：[[状态]]

状态与时间

状态与时间的相互关系是非常重要的。我发现了一个很有意思的现象，我已经脱离了过去那个单个分别解释的时代，现在我所处的这个阶段，它们之间的联系是至关重要的，我必须搞清楚它们如何制约彼此，然后我才能进入下一个阶段，从更宏观的尺度探讨整个系统。

对能量的探究

发表于 2023-01-14

2022.1.14 第一次

能量包括些什么？

注意力能量

运用这段讨论作为一个简要的概述：“注意力模式的开启。可以这么比喻：我们的注意力的门是两个石头，两个石头呈现一种相向运动的趋势，直到贴合在一起；也就是说这个门是自发关闭的；接收信息，首先你要把这个门打开，然后信息怎么进来还不一定呢，什么时候进来还不一定呢，你无法做到把所有的能量都用到让信息进入的那一小段时间里，可以说，你必须得一直开门，而且如果你细算，你大部分的时间都是浪费的，这个时间或许乘以的是一个系数（单位时间能量损耗），意味着你大部分的能量是浪费的。也许这便是为什么流量值钱的一个原因，它可以束缚住这些愚蠢的人们（包括我在内）。”

体力

这个在我的生活里是与足球等运动相挂钩的。我现在对它的一种关注，包括：

  • 我为了踢比赛的时候有更多的体能，我每天应该花多少[[时间]]锻炼？

  • 体力的消耗到了什么样的程度，会对其他的能量形式造成影响？

  • 体力的改进，会如何影响其他两种能量的改进？

我昨天刚手淫完，得过几天再开始运动，就明天开始吧。（[[待办事项]]）

信息能量

我每天能存储的信息是有限的。对于结构上的有限性我还上不清晰，但是我清晰的是：它对内存有限，而且这个内存是动态分配的，是需要运用能量完成分配内存，然后把一整个信息转运过去。

这里面存在着几个很有意思的关键词。

一整个信息。一整个信息值得其实不是一个完整的信息，而是我首先有一些关注的点，我把这些点提炼出来，由这些点组成一个“完整的信息”。

我平时刷懂球帝，刷微博，刷朋友圈，刷知乎，我不动脑子都能想到他们的信息会是什么样的结构，而且我深知是狗屎，变得不过时那些个信息具体的值，为了这个我还要分配我的信息能量，未免有些太傻逼。

而且信息能量是与注意力能量直接挂钩的。我们首先得开启注意力模式，才能接受新的模式，然后我要谈一谈这个注意力模式的开启。

注意力模式的开启。可以这么比喻：我们的注意力的门是两个石头，两个石头呈现一种相向运动的趋势，直到贴合在一起；也就是说这个门是自发关闭的；接收信息，首先你要把这个门打开，然后信息怎么进来还不一定呢，什么时候进来还不一定呢，你无法做到把所有的能量都用到让信息进入的那一小段时间里，可以说，你必须得一直开门，而且如果你细算，你大部分的时间都是浪费的，这个时间或许乘以的是一个系数（单位时间能量损耗），意味着你大部分的能量是浪费的。也许这便是为什么流量值钱的一个原因，它可以束缚住这些愚蠢的人们（包括我在内）。

刷meme是要花很多时间的，而且大差不差，对我也没什么帮助，但是研究meme的结构就不一样了，这是一种分析学，还挺有意思，我以后可以试一试。

思考能量

这个过程，提取大量的可能信息（扩大决策空间），不断地转化自己的视角（模式选择），在相对确定模式之后，需要进行推理（具体模式推断），得出结果，这个时候有时我还会带入这个结果去检验是否正确（查验），如此循环往复。

其中这个扩大决策空间是比较费力的，可以说是最费力的；然后这个模式选择，相对没有那么费力，在之后的具体模式推断和查验到都还好。所以说，我需要解构这个“扩大决策空间”的过程，然后给它限定边界，给它一些预设的（在算法上必须是绝对正确的）引导。而且我每个具体的都得这样，只有例子足够多，我才能进行合理的抽象分析。这个摄入例子就是学习啊，这就是我为什么必须要学习。

糟糕的，混乱的，愚蠢的思考是非常消耗的能量。也就是说，如果要完成一系列既定的任务，它们本质上大差不差，那这个时候我其实需要优化我的思考过程，比如确定决策空间的基本模型，然后先有几个基本的模式，每个模式里的模式推断要很熟悉，这个一般需要大量的重复，但是重复有可以的和不经意的，有的时候搭配一下更好。

能量的恢复

睡眠

完成的睡眠。6-8小时的睡眠，这个时候我的大脑能够得到充分的休息。

小睡。一般15-30分钟，睡完之后能够继续进行数学学习。

中睡。一般1-2小时，睡完以后大脑一片浆糊，适合出去玩，有时还会影响晚上的睡眠。

“体育锻炼”

他们说体育是恢复能量的一种方式，这个从结果的现象来看，是有关联的，但是，不能就此止步啊，是真的是因为我们体育锻炼，那种能量的消耗，使得我们恢复吗？

我有猜想，尚未被证实，就是：是在进行体育运动的时候，我们精力去思考，去摄入太多的信息，我们把所有的权力，交给我们的肌体本身。

一般的足球，野球，踢完以后很舒服，但正式比赛就不一样了，踢完像死了个人。

所以真正恢复能量的，应该是这种对“注意”与“思考”活动的隔绝，的置之不理。

这样来看，锻炼自己放空，发呆，未免不更合适一些。

我现在致力于把自己的意识，自己的精神现象用语言描绘出来，尽可能本真地还原那些过程；用简洁的话来讲，就是我都在想些什么？我怎么想这些东西？每一个过程之中我都是从什么决策空间之中以什么样的模式做出决策？可以说，我现在要开始拆解我的意识，拆解我的思维，我要对我的认知模式有一个根本上的认识，当我成功的对它们做出界定，分类，归纳，对一整个系统，或者说生态有确定性的认识之后，我就可以进行一种重设；可以理解为：我要进行改造，这样说，其实是不贴切的，倒不如说，我准备控制自己的意识，从盲目的顺从它的流向，到获得这个流向的控制权。

事实上，这是极为抽象的，而且我的意识经常不可避免的走向一种简单的，舒适的，毫无意义的循环之中，它以这种方式完成对时间的消耗；贪污啊，腐败啊，腐烂的意识啊！

不难意识到，我这篇文章其实带有一种导论的性质，或者说，它是一种实验猜想，我并不知道这个实验能不能成功，但我起码有一个设计，并且，从明天开始，我就要进行这个实验。其实，这些天我已经开始与我的意识流打交道，结果不太好，我每天都过的很混乱，我找不到一个模式，我想，可能我需要先让我的意识朝向一个既定的方向流动，然后它们还得是一种稳定的流动，坚不可摧的流动，然后我在我自己的意识中，成为一个置身事外的观察者；光是想想，就充满了兴奋。

明天，我就要开始记录啦！

我现在语言的同质化太过严重，有没有一种可能，人们经常用的语言结构会决定他们的思维结构；如果这个猜想是真的的话，我必须要学习更多的语言表述方式，如果说，人格分类真的有它的可取之处，那我可能需要每一个人格都找几个典型作家，学习他们的表述方式，我比较感兴趣的有INTP, INTJ, INFP, INFJ，如果要找的话，米兰昆德拉，intj我不知道有谁，加缪，陀思妥耶夫斯基。听起来就很幼稚，我承认这只是一个优质的搭建，如果在我实践的过程中意识到了它的愚蠢所在，我会及时停止的。

有的时候，语言真的会决定思想的极限，再抽象一步，语言其实是一种符号，它包括了这些符号之间的关系，真正重要的是这些关系的结构，也就是说，一个人脑子里的抽象结构决定了他能诞生什么样的思想，决定了他思想的极限。也并不是说，一定要学数学才能变的有思想，只是我感觉，学习数学是触及这种抽象的一条捷径。有些愚蠢的人，很容易被那些具象的信息淹没，我不是否认那些信息的价值，但如果一个人的生活，他的眼里只有那些具象的，触手可及的，未免有些可悲，不完善。当然，我也意识到我的生活有很多的缺失，我的五感其实都快退化了，我平常基本不需要它们，这样也不好。

我其实并没有浪费十九年的时间，我用了十九年的时间演绎人的愚蠢，对于愚蠢，我拥有十九年的经验，意味着，我对愚蠢有一种非常强的熟悉，这对我后续的一些推导十分有帮助，一旦我感受到什么结构非常相似，拿它有两种可能，一是它非常基本，二是它非常愚蠢，这可以作为一种搜索的边界条件，可以帮我剪枝。

我现在是在一种入门的阶段，倒不如说，进入黑暗，然后获得一种抽象的视力，也许我之后的人生会一次又一次的进入这种情景，我不知道，但我只知道，我要一直抽象，一直抽象，一直抽象，一直抽象，一直抽象，一直抽象。

我并不知道我最终会走到哪里，只是现在我的心里有一个明确的方向，我确信我应该这样走下去。它不一定是对的路，甚至最终可能会毁灭我；但这是我用我十九年的过去所确定的方向，也只有顺延着这个方向，我才是我自己，这样走下去，我才有可能形成一些突破。

我记得我过去的所有的思考：我当时是怎么想的，是什么因素，什么模式致使我这么想的；我现在清楚地看到它们的局限性。但是，时常我也是仅仅看到它的局限性，我有所构想，构想如何去解决它们，但是，我并没有一种成功的经历，可以为我提供真正值得参考的思路；我需要读书，需要与人交谈，需要去接触这个世界；可是，我又深知自己的愚蠢，深感自己的渺小，而且当我真正的要和一个人或者事物进行交流，我需要付出很多很多时间与精力，为他们构建一个转译的系统，走进我的内心世界；我不确定什么值得我花费这些时间与精力，我又没有魄力去直接随便的找一个切入口，有一种对仪式的执恋，从宏观上的庄严束缚着我的手脚；它其实一直在束缚我，我能够意识到它，但还没有强大到去解构它。

我的生活充满了迷茫与困惑，太多的信息，太多的可能展开的未来充斥了我的整个脑海，我很累，某种意义上有些痛苦。

我不确定我的明天是什么样的，我更无法确定我的后天又会如何；我仅仅记得，随时我都有可能会否认我之前所有的结论，但过不了多久，我又会回到同样的出发点；它致使我对人若即若离，它致使我变得捉摸不透。

我渴望永恒的生命，永恒的存在，她自身足够强大，她又愿意给我分一片世界，分一些时间；任何时间，任何地点，发生什么事情都不重要，我们随时随刻都可以回到上一次保存的那种状态。这未免有些太抽象了，我对马尔克斯对爱情的理解的理解非常深刻，这是一种领悟，但是我写不出来，我也不知道如何和别人去讲。

我只能靠我自己啊。
